Commandr

Web, meet Speech.

Commandr is a library that makes it simple for developers (and eventually page visitors) to register custom speech commands. Commandr is also a two way street, it provides the simplest text-to-speech interface imaginable. And the whole thing is powered by IBM Watson!

Step 1:


<script type="text/javascript" src="https://commandr.me/commandr.js"></script>

Step 2:


//All commands are preceeded by the word "Commander"
//When the user says "Commander introduction" the registered function is run. The function could do anything.
Commandr.register("introduction", function() {
  //In this case, the function will call Commandr.speak which will audibly say "hello world!" to the user.
  Commandr.speak("hello world!");
});

Why Use It?

Commandr makes it easier and more fun to not only navigate, but truly use the web. It empowers developers to create a more meaningful, interactive experience for their page visitors.
